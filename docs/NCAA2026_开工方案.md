## 二、主线一：Raddar（优先）

### 2.1 数据流概览

```
M/WRegularSeasonDetailedResults (历史+2026)
    → prepare_data 对称化
    → Elo + GLM quality + seeds 特征
    → Leave-one-season-out XGBoost
    → Spline 校准
    → 预测 SampleSubmissionStage2 中所有 (team1, team2)
    → (可选) set_optimalStrategy 覆盖部分风险队
    → submission.csv
```

### 2.2 参考实现

- **主要参考**：`notebooks/reference/vilnius-ncaa.ipynb`（完整 Raddar 流程）
- **策略覆盖参考**：`notebooks/reference/ncaa2025-7th-solution.ipynb`（Raddar + set_optimalStrategy）

### 2.3 已有模块

| 模块 | 路径 | 用途 |
|------|------|------|
| prepare_data | `src/raddar_utils.py` | Detailed Results 对称化 |
| get_roundOfMatch, set_optimalStrategy | `src/optimal_strategy.py` | bracket 推断、33.3% 策略覆盖 |

### 2.4 实施步骤（供代码编写参考）

1. **数据加载**
   - 目录：`march-machine-learning-mania-2026/`
   - 需加载：MRegularSeasonDetailedResults, WRegularSeasonDetailedResults, MNCAATourneyCompactResults, WNCAATourneyCompactResults, MNCAATourneySeeds, WNCAATourneySeeds, MSeasons, 等

2. **特征工程**
   - prepare_data → Elo → GLM quality → seeds（参考 vilnius 特征列表）
   - 构建 X：所有 2026 对阵（来自 SampleSubmissionStage2 或自生成）

3. **模型**
   - Leave-one-season-out XGBoost（预测 margin）
   - Spline 校准 margin → 概率

4. **预测**
   - 对 X 中每行预测 Pred，输出 ID + Pred

5. **后处理（可选）**
   - 调用 set_optimalStrategy，选择 2026 年 risk team（如往年 Drake）

### 2.5 输出

- `submission_raddar.csv`：ID, Pred

## 三、主线二：goto_conversion（待概率表更新）

### 3.1 数据流概览

```
Substack 概率表 (Team, rd1_win~rd6_win)
    → 队名 → TeamID 映射 (W/MTeamSpellings)
    → 对 SampleSubmissionStage2 每行：
        round = get_roundOfMatch(team1, team2, seeds_df)
        P(win|reach) = rd{r}_win / rd{r-1}_win
        odds = [1/P1, 1/P2]
        [p1, p2] = preprocessed_goto_conversion(odds)
        Pred = p1 (若 team1 < team2)
    → (可选) set_optimalStrategy
    → submission.csv
```

### 3.2 参考实现

- **主要参考**：`notebooks/reference/goto-conversion-winning-solution.ipynb`（preprocessed_goto_conversion、P(win|reach) 公式）
- **结构参考**：`notebooks/reference/updated-goto-conversion-winning-solution.ipynb`（加载 538data、set_optimalStrategy）
