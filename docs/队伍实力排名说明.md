# 队伍实力排名说明

## 一、排名来源

### 1. Pairwise 排名（主参考）

**数据**：`matchupProbabilities_men.csv` / `matchupProbabilities_women.csv`（goto_conversion 2026 两两对阵 Pred）

**算法**：对每支队伍计算**预期胜场**（Expected Wins）

- 若该队为 T1：该行 Pred 即 P(该队胜)
- 若该队为 T2：该行 1−Pred 即 P(该队胜)
- 预期胜场 = 对所有 67 个对手的上述概率求和
- 范围约 0–67，越高表示模型认为越强

**输出**：`docs/team_ranking_men_pairwise.csv`、`docs/team_ranking_women_pairwise.csv`

### 2. Projection 表排名（参考用）

**数据**：`mensProbabilitiesTable*.csv` / `womensProbabilitiesTable*.csv`（按轮次晋级概率）

**算法**：按 **rd6_win**（夺冠概率）降序；可选按 **Elo_Rating** 降序

- rd6_win：夺冠概率，反映在 bracket 路径下走到最后的概率
- 排除占位符（Y16、W16、X11 等）

**输出**：`docs/team_ranking_men_projection.csv`、`docs/team_ranking_women_projection.csv`

## 二、能否用 Projection 表做「自上而下」参考？

**可以，但要理解差异：**

| 维度 | Pairwise 排名 | Projection 排名 |
|------|---------------|-----------------|
| 含义 | 与所有对手「虚拟循环赛」的预期胜场 | 在特定 bracket 下夺冠的概率 |
| 是否依赖 bracket | 否 | 是 |
| 是否受路径影响 | 否 | 是（如分区强弱、赛程） |
| 用途 | 队伍绝对实力 | 在当前对阵结构下走到最后的概率 |

**建议用法**：

1. **Pairwise 排名**：更接近「实力」排序，适合识别被低估/高估的队伍。
2. **Projection 排名**：作为「在现有 bracket 下谁更可能夺冠」的参考；若 bracket 变化，需重新生成。

若你手上有 **2026 projection 表**，可将脚本中的路径改为 2026 文件，以得到与今年 bracket 一致的 projection 排名。
